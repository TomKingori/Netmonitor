# Network Monitoring App

## Setup and installation

Make sure you have the latest version of Python installed.

Install MongoDB

Install RabbitMQ Server

```sh
git clone <repo-url>
```

```sh
pip install -r requirements.txt
```

## Running the app

On one terminal
```sh
./run-server.sh
```

On another terminal
```sh
./run-all.sh
```

## Viewing The App
Go to `http://127.0.0.1:5000`
